# 🤖 WhatsApp Image Agent

Agente inteligente desarrollado en Python capaz de analizar imágenes recibidas y generar respuestas automáticas utilizando Inteligencia Artificial.

## 🚀 Descripción

Este proyecto implementa un agente inteligente que procesa imágenes enviadas a través de WhatsApp, analiza su contenido mediante modelos de IA y genera respuestas basadas en la información detectada.

El objetivo es demostrar habilidades en:

- Python
- Inteligencia Artificial
- Procesamiento de imágenes
- Automatización de tareas
- Integración de APIs
- Desarrollo de agentes inteligentes

## 📂 Estructura del proyecto

```bash
whatsapp-image-agent/
│
├── .venv         # credenciales
├── agent.py      # Lógica principal del agente
├── main.py       # Punto de entrada de la aplicación
├── foto.jpg      # Imagen de ejemplo
└── README.md
```

## ⚙️ Tecnologías utilizadas

- Python 3.x
- OpenAI API (si aplica)
- Pillow / OpenCV (si aplica)
- Requests
- WhatsApp API
- Inteligencia Artificial Generativa

## 🔥 Funcionalidades

- Recepción de imágenes.
- Análisis automático de contenido visual.
- Interpretación mediante IA.
- Generación de respuestas inteligentes.
- Arquitectura modular y extensible.

Entrada:

Usuario envía una imagen por WhatsApp

## 🛠 Instalación

Clonar repositorio:

```bash
git clone https://github.com/TU-USUARIO/whatsapp-image-agent.git
```

Entrar al proyecto:

```bash
cd whatsapp-image-agent
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar:

```bash
python main.py
```
